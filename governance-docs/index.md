# Governance

This suite sets out how the From Wildfires to Regeneration Framework is directed, how communities and partners participate in it, how it sustains itself, how it manages risk, and how it protects and shares its intellectual property as it is adopted more widely.

All documents in this suite are **Version 0.1 Working Drafts**. They establish structure and scope only, ahead of substantive drafting, and are intended for iterative refinement rather than immediate publication or external distribution.
