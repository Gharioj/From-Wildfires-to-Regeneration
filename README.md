# From Wildfires to Regeneration

**Version 1.0 — Published**

## About

From Wildfires to Regeneration is a framework for moving from wildfire crisis toward long-term ecological and social regeneration. This repository holds the Framework's Version 1.0 publication suite: its founding manuscript, its capability structure, and the materials prepared for governance, patron, and strategic-partner audiences.

*Note: the descriptions below are based on each document's own title and role, not a full reading of its contents — refer to the documents themselves as the authoritative source.*
