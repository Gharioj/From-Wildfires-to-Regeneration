# About this site

This website is the official public documentation portal for the **From Wildfires to Regeneration** Framework. It is generated from, and reads directly from, the [`From-Wildfires-to-Regeneration`](https://github.com/{{ site.repository }}) GitHub repository, which remains the authoritative source for every document presented here.

## Authorship

The Framework's Founding Manuscript, *Dagli Incendi alla Rigenerazione*, was authored by **Ghari o'Janjim**, an independent researcher in territorial regeneration.

## Repository structure

- **`Publication/`** — the Version 1.0 publication suite and the Founding Manuscript, in Microsoft Word (`.docx`) format. These are the authoritative masters.
- **`Publication/IT/`** — the Italian Version 1.0 translation suite.
- **`Governance/`** — the governance documentation suite (Version 0.1, working draft), in Markdown.
- **`Governance/IT/`** — the Italian translation of the governance documentation suite.

## Document status

- **Publications:** Version 1.0 — Published.
- **Governance:** Version 0.1 — Working Draft. These documents establish structure and scope ahead of substantive drafting, and are intended for iterative refinement rather than immediate publication or external distribution.

## A note on this website

Publications are distributed as Word documents. To present them as readable web pages without altering the originals, this site includes a separate, generated plain-text rendering of each publication's content alongside a direct download link to the original `.docx` file. The governance pages render the Markdown source files from the repository directly. If a source document is revised, its corresponding page on this site should be regenerated to match.

Nothing in `Publication/` or `Governance/` is modified by the existence of this website — every original file remains exactly as published in the repository.
