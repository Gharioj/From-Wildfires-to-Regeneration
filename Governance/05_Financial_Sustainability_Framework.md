# Financial Sustainability Framework

**Status:** Version 0.1 Working Draft

## Purpose

To set out the principles guiding the long-term financial sustainability of the From Wildfires to Regeneration Framework, including funding, resource allocation and financial stewardship.

## Scope

Covers funding and revenue sources, resource allocation, and financial oversight and sustainability planning. Does not include detailed financial projections or budgets.
